## ◈ The Origin Story ◈

</div>

This is **Project #01** — the one that started it all.

When I began learning Python, the classic first real project everyone builds is a To-Do List. Not because it's flashy. But because it forces you to think about **data**, **user input**, **loops**, and **functions** — all at once, for the first time.

This project was messy, had bugs, got rewritten twice, and I'm still proud of it. It's a snapshot of Day 1.

---

<div align="center">

## ◈ What It Does ◈

</div>

```
┌─────────────────────────────────────────┐
│           📋  MY TO-DO LIST             │
├─────────────────────────────────────────┤
│                                         │
│   [1]  Add a task                       │
│   [2]  View all tasks                   │
│   [3]  Mark a task as done  ✔           │
│   [4]  Delete a task        ✘           │
│   [5]  Exit                             │
│                                         │
└─────────────────────────────────────────┘
```

A clean command-line interface to manage your tasks:

| Feature | Description |
|---------|-------------|
| ➕ Add Task | Add any task with a title |
| 📋 View Tasks | See all pending and completed tasks |
| ✅ Mark as Done | Check off completed tasks |
| 🗑️ Delete Task | Remove tasks you no longer need |
| 💾 Persistent Storage | Tasks saved to a file — survive program restarts |

  ├── Functions — breaking logic into reusable blocks
  ├── Loops — while loops for the menu, for loops for display

## ◈ Getting Started ◈

</div>

### Prerequisites

All you need is Python 3. No external libraries required.

```bash
python --version   # Should be 3.x
```

### Run It

```bash
# Clone the repo
git clone https://github.com/<your-username>/todo-list.git

# Navigate into the folder
cd todo-list

# Run the app
python todo.py
```
